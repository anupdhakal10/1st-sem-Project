🎬 Online Movie Ticket Booking System
------------------------------------------------------------------------------

An Online Movie Ticket Booking System developed as our first team project during the 1st Semester of the Bachelor of Computer Applications (BCA). This project simulates the online movie ticket booking process with an interactive and user-friendly interface.

✨ Features
-------------------------------------------------------------------------------
Browse available movies
View showtimes
Select seats
Demo payment system
Booking confirmation
Admin dashboard for movie management
Responsive design

🛠️ Technologies Used
---------------------------------------------------------------------------------
HTML5
CSS3
JavaScript (Vanilla JS)
Local Storage

🚀 How to Run
----------------------------------------------------------------------------------
Clone or download the repository.
Open index.html in your web browser.
